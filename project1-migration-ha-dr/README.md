# 🚀 Project 1: Multi-Cloud DB Migration with HA & DR

## 📌 Overview
This project demonstrates migration of a local MySQL database to AWS RDS with High Availability, Read Replicas, and Disaster Recovery setup.

## 🛠️ Technologies Used
- MySQL
- AWS RDS
- GCP BigQuery
- SQL

## ⚙️ Architecture
Local MySQL → AWS RDS → Read Replica → Backup → BigQuery

## 🚀 Steps Implemented
1. Created local MySQL database
2. Migrated data to AWS RDS
3. Configured Multi-AZ deployment
4. Created read replica
5. Implemented automated backups
6. Performed restore testing
7. Exported data to BigQuery

## 📊 Results
- Achieved high availability
- Tested failover successfully
- Enabled analytics using BigQuery

## 🧠 Key Learnings
- Database migration
- High availability architecture
- Disaster recovery strategies
- Multi-cloud integration
